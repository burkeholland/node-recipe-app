# Recipe App - CSS Framework Mock Designs

This folder contains 5 complete redesigns of the Recipe App using the most popular CSS frameworks.

## How to Preview

Simply open any of the HTML files in your web browser. Each file is completely self-contained with all necessary CSS and JavaScript loaded from CDNs.

## Mock Designs

### 1. Bootstrap (mock1-bootstrap.html)
- **Framework**: Bootstrap 5.3.2
- **Style**: Clean, professional design with cards and modals
- **Features**: Responsive grid, gradient hero section, hover effects
- **Icons**: Bootstrap Icons
- **Color Scheme**: Dark navbar with purple gradients

### 2. Tailwind CSS (mock2-tailwind.html)
- **Framework**: Tailwind CSS (via CDN)
- **Style**: Modern, utility-first design with vibrant colors
- **Features**: Gradient backgrounds, floating animations, rounded elements
- **Icons**: Custom SVG icons
- **Color Scheme**: Purple/indigo gradients with colorful recipe cards

### 3. Material Design (mock3-material.html)
- **Framework**: Custom Material Design implementation
- **Style**: Google Material Design principles
- **Features**: Elevation shadows, FAB button, clean typography
- **Icons**: Material Icons
- **Color Scheme**: Blue theme with card-based layout

### 4. Bulma (mock4-bulma.html)
- **Framework**: Bulma 0.9.4
- **Style**: Clean, modern CSS framework with flexbox
- **Features**: Hero sections, level components, modals
- **Icons**: Font Awesome 6
- **Color Scheme**: Teal/turquoise primary color

### 5. Foundation (mock5-foundation.html)
- **Framework**: Foundation 6.8.1
- **Style**: Professional, enterprise-focused design
- **Features**: Top bar navigation, callouts, reveal modals
- **Icons**: Font Awesome 6
- **Color Scheme**: Blue/gray professional palette

## Framework Comparison

| Framework | Bundle Size | Learning Curve | Customization | Best For |
|-----------|-------------|----------------|---------------|----------|
| Bootstrap | Medium | Easy | Good | General purpose |
| Tailwind | Small* | Medium | Excellent | Modern apps |
| Material | Medium | Easy | Limited | Google-style apps |
| Bulma | Small | Easy | Good | Clean designs |
| Foundation | Large | Medium | Excellent | Enterprise apps |

*When using the CDN version, the bundle is large. In production with PurgeCSS, it's very small.
